Um projeto realizando todo o trabalho de data science em um banco de dados do Keggle sobre o consumo de álcool em alunos de Português e Matemática.

Foi realizada a limpeza e análise dos dados ademais a elaboração de um modelo preditivo sobre os dados escolhidos.

Além de um Dashboard em Power BI sobre as features relevantes.

O objetivo do modelo é auxiliar no combate ao consumo de álcool entre os jovens da escola de onde pertence os dados. Prevendo a chance de um aluno consumir ou não grandes quantidades de álcool com base em suas informações básicas coletadas pela escola.
