Um projeto realizando todo o trabalho de data science em um banco de dados do Keggle sobre o consumo de álcool em alunos de Português e Matemática.Foi realizada a limpeza e análise dos dados ademais a elaboração de um modelo preditivo sobre os dados escolhidos,além de um Dashboard em Power BI sobre as features relevantes.
O objetivo do modelo é auxiliar no combate ao consumo de álcool entre os jovens da escola de onde pertence os dados,prevendo a chance de um aluno consumir ou não grandes quantidades de álcool com base em suas informações básicas coletadas pela escola. 

O dashboard Power BI se encontra no link: https://app.powerbi.com/links/qXFoinZfdb?ctid=ea41a2d9-e85a-4edf-a41a-942f7815d17d&pbi_source=linkShare | Ou no arquivo alcool.pbix.
O nootebook com todo o processo de análise,limpeza e manipulação de dados se encontra notebook Alcool.ipnyb.

Dados obtidos em:
https://www.kaggle.com/datasets/uciml/student-alcohol-consumption
